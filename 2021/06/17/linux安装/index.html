安装软件都要先有安装包，可以下载一个压缩包，然后解压到你指定的文件夹就好了

解压命令：

tar -xvf 压缩包的名字 -C 你指定的目录

例子：
tar -xvf jdk-8u261-linux-x64.tar.gz -C /usr/lib/jvm

# 安装 JDK

安装之前你要检测系统有没有 JDK 默认的安装包

rpm -qa | grep jdk　(命令的意思是搜索 jdk，查看安装包文件)

卸载 OpenJDK

rpm -e --nodeps 名称　　　　或者　　　　yum remove openjdk

  1. 找到压缩包
  2. 解压文件

tar -xvf jdk-8u261-linux-x64.tar.gz -C /usr/local

  1. 配置变量环境
    Linux 环境变量配置都在：/etc/profile 文件中，通过以下命令进入编辑

vim /etc/profile

or

vi /etc/profile

export JAVA_HOME=/usr/local/jdk             # usr/local/jdk (软件的目录)
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
  1. 编辑好后保存退出 (先按 ESC 键)

:wq

如果有问题

:q! (强制退出没保存！！！)

  1. 更新环境环境变量

source /etc/profile
